# Core Data Structures: Binary Search Tree (C++)

This project archives my early academic work with core computer science fundamentals. Originally developed in a local Code::Blocks environment and tested via online compilers (Ideone), this repository contains a custom implementation of a Binary Search Tree (BST) built from scratch.

* **Focus:** Object-Oriented Programming, Memory Management, Pointer Manipulation, and algorithmic efficiency ($O(\log n)$ time complexity).
* **Language:** C++

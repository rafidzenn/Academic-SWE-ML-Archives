# Software Lifecycle & Version Control (Bitbucket) Fundamentals

This directory contains my conceptual notes and programmatic simulations of standard software development methodologies and enterprise version control systems. 

Instead of just writing documentation, I built Python-based Object-Oriented models to simulate how these frameworks operate. 

* **Part 1: SDLC:** Explores Agile/Waterfall methodologies, complete with an interactive state-machine simulation.
* **Part 2: Bitbucket Workflows:** Documents branching strategies and models a Pull Request lifecycle using OOP principles.
